Usando Links Externos

Valores dos Parâmetros

  1. target
    • _blank = abrir o link em outra janela
    • _self = abrir o site sobrepondo o seu(padrão)
    • _top
    • _parents
  2. rel
    • next = link vai abrir a proxima parte do documento
    • prev = link vai abrir a parte anterior do documento
    • author = o link vai abrir o site do autor atual
    • external = o link é de outro site totalmente diferente do atual
    • nofollow = usado para patrocinios|para o site não se perpetar|.
